2. Proyecto

Nombre: Diseño e Implementación de una Estación Meteorológica con conectividad IoT como recurso didáctico en una institución educativa en Arequipa. :contentReference[oaicite:1]{index=1}

2.1 Equipos y materiales
  • ESP32, Raspberry Pi 4
  • Sensores: viento (dirección/velocidad), temperatura, humedad relativa, radiación UV
  • Fuente DC, caja estanca IP65, multímetro, cables
  • Computadora con software de programación

3. Diseño del escudo de radiación

Se modeló el escudo de radiación en SolidWorks para proteger el sensor térmico de la radiación directa y mejorar la calidad de las mediciones. :contentReference[oaicite:2]{index=2}

4. Impresora 3D Creality

Se empleará la Creality Ender-3 V3 KE (FDM) con volumen de impresión 220 × 220 × 240 mm, adecuada para fabricar las piezas del proyecto. :contentReference[oaicite:3]{index=3}

4.1 Principales características
  • Build volume: 220 × 220 × 240 mm
  • Dimensiones: 433 × 366 × 490 mm
  • Peso neto: 7.8 kg
  • Velocidad máx.: 500 mm/s (típica: 300 mm/s)
  • Aceleración máx.: 8000 mm/s²
Software de laminado

Se verificó el uso de Creality Print (V4.3.8) para preparar los modelos y generar G-code. :contentReference[oaicite:4]{index=4}

5. Observaciones

  • Se confirmó el volumen de impresión y características técnicas de la Ender-3 V3 KE.
  • Se validó la disponibilidad de Creality Print como herramienta de laminado principal.
  • Las capacidades del equipo son adecuadas para fabricar el escudo de radiación diseñado.

6. Conclusiones

  • Queda definida la base técnica de hardware + software para la etapa de prototipado.
  • El diseño del escudo está listo para preparación e impresión.
  • El RACI clarifica responsabilidades del equipo para la fabricación.
