Estación Meteorológica MeteoLab

Proyecto de innovación que busca soluciones educativas, dirigido a estudiantes de secundaria, institutos y universidades.

¿Qué encontrarás aquí?

Este sitio reúne toda la información relacionada con el proyecto desarrollado para el curso de Investigación e Innovación Tecnológica. El proyecto consiste en el diseño y construcción de una estación meteorológica. Utiliza el menú superior para explorar las diferentes secciones y conocer más sobre el desarrollo, funcionamiento y resultados del proyecto.

Cómo se actualiza

Las lecturas son tomadas mediante los sensores implementados y enviadas a través de la red WiFi hacia nuestra base de datos en el canal de ThingSpeak. Puedes visualizar estos datos en la pestaña “Datos”. Aquí te indicamos la última vez que llegó un registro.

Si ves “hace X s”, significa el tiempo transcurrido desde la última muestra.

Acerca de este proyecto

El presente proyecto consiste en el desarrollo e implementación de una estación meteorológica enfocada en la medición de variables como temperatura, humedad, viento (dirección y velocidad), radiación solar y presión atmosférica. Su propósito principal es servir como un recurso didáctico para estudiantes. Esta plataforma web está diseñada para facilitar la consulta ágil de los datos monitoreados, los cuales se visualizan en tiempo real.
